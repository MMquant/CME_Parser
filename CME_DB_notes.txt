CME data database notes

1. PostgreSQL install
apt-get install postgresql postgresql-client
sudo -u postgres bash
# bash client start
psql
# postgres pw change
psql: \password postgres
# adding system user r_client
sudo adduser r_client
# create db user
sudo -u postgres createuser r_client
# create database
createdb -O r_client cme_data
## system user and db user must be the same to successful connect !!!
